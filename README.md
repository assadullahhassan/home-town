# Visit Burundi - Tourism Landing Page

A visually engaging, landing page built to showcase the natural beauty and cultural heritage of Burundi. This project highlights the "Triple Crown" of Burundian identity: the Source of the Nile, the "Land of a Thousand Hills," and the Royal Drummers of Gishora.
